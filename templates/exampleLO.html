Logistic Regression Practical Example
Problem Description

This example uses Logistic Regression to classify employee burnout levels.

The goal is to determine whether an employee has High or Low burnout risk based on workplace and behavioral variables.

Classification models like this are widely used in HR analytics, psychology research, and workforce monitoring.

Dataset Explanation

Dataset used: Mental Health Burnout Dataset

The dataset contains employee indicators such as:

  • Working hours
  • Mental fatigue score
  • Company type
  • Remote work status
  • Stress indicators

Target variable:

  • 0 → Low Burnout
  • 1 → High Burnout
Data Preparation
  • Removing Medium burnout category
  • Removing missing values
  • Encoding categorical variables
  • Feature scaling
  • Train/Test split
