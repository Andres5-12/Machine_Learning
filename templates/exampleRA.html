Random Forest - Application Example
Problem Description

This project aims to predict employee burnout levels using a Random Forest classification model.

The objective is to classify employees into two categories:

  • Low Burnout
  • High Burnout

This type of model helps organizations identify at-risk employees and take preventive actions.

Dataset Explanation

Dataset used: Mental Health Burnout Dataset

The dataset contains information about employee work conditions and mental health indicators.

Variables Used

Independent Variables:

  • Work hours
  • Stress level
  • Mental fatigue score
  • Work environment

Dependent Variable:

  • Burnout Level (0 = Low, 1 = High)
Data Preparation
  • Removed "Medium" burnout category
  • Handled missing values
  • Encoded categorical variables
  • Split dataset into training and testing sets
  • Scaled numerical features
Model Training

A Random Forest Classifier was trained using multiple decision trees to improve prediction accuracy.

The model learns patterns from the training data and applies them to unseen data.
