Linear Regression Practical Example
Problem Description

The problem addressed in this example is the prediction of stock prices using Machine Learning techniques. Financial markets generate large amounts of historical data that can be analyzed to detect patterns and trends.

Predicting stock prices is important because it can help investors, analysts, and financial institutions make better decisions based on data instead of intuition.

In this case, a Linear Regression model is used to learn the relationship between market indicators and the closing price of a stock.

The objective is to estimate the closing price based on historical variables.

Dataset Explanation

The dataset used in this example contains historical financial information from the stock market.

Dataset used: Tesla Stock Price Dataset (NASDAQ TSLA historical data)

It includes daily records of stock activity such as prices and trading volume.

This dataset allows the model to learn how different financial indicators affect the stock price.

Variables Used

Independent variable:

  • Open price

Dependent variable:

  • Close price
Data Preparation
  • Removing null values
  • Selecting relevant variables
  • Splitting data into training and testing sets
  • Training the regression model
