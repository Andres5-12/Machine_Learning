Linear Regression Concepts
What is Linear Regression

Linear Regression is a supervised Machine Learning algorithm used to predict a numerical value by modeling the relationship between variables using a straight line.

The main goal of linear regression is to find the mathematical relationship between variables and use it to make predictions.

Y = mX + b

Where:

  • Y is the dependent variable
  • X is the independent variable
  • m is the slope of the line
  • b is the intercept

The algorithm finds the best possible line that minimizes the difference between real values and predicted values.

Independent and Dependent Variables
Independent variables

Independent variables are the variables used to explain or predict another variable.

They are also called features.

These variables act as the input of the model.

Example:

  • Size
  • Location
  • Number of rooms
Dependent variable

The dependent variable is the value we want to predict.

It is also called the target variable.

This variable depends on the independent variables.

Example:

House price would be the dependent variable.

Training Data

Training data is the dataset used to train the Machine Learning model.

This data contains examples where the correct answer is already known.

The model learns patterns and relationships between the independent variables and the dependent variable.

The training process allows the model to:

  • Learn patterns
  • Detect relationships
  • Reduce prediction error
  • Generalize to new data

Good training data is important because the model performance depends on data quality.

Regression Line

The regression line is the straight line that best represents the relationship between the independent and dependent variables.

This line is calculated using mathematical optimization techniques like the Least Squares Method.

The objective is to minimize the total prediction error.

The regression line shows:

  • The trend of the data
  • The relationship between variables
  • The prediction behavior

If the line goes upward, the relationship is positive.

If the line goes downward, the relationship is negative.

Slope and Intercept
Slope

The slope represents how much the dependent variable changes when the independent variable increases by one unit.

It shows the strength and direction of the relationship.

If the slope is positive:

The dependent variable increases.

If the slope is negative:

The dependent variable decreases.

Intercept

The intercept is the point where the regression line crosses the Y axis.

It represents the predicted value when the independent variable equals zero.

It can be interpreted as the starting value of the model.
