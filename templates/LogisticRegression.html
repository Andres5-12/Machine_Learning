Logistic Regression Concepts
What is Logistic Regression

Logistic Regression is a supervised Machine Learning algorithm used for classification problems, where the output is a category rather than a numerical value.

Its main objective is to estimate the probability that a given input belongs to a specific class.

f(x) = 1 / (1 + e-x)

Where:

  • f(x) represents the probability
  • e is Euler's number
  • x is the linear combination of input variables

The output is a value between 0 and 1, which can be interpreted as probability.

Independent and Dependent Variables
Independent Variables

Independent variables are the inputs used to make predictions.

They are also called features.

These variables represent the characteristics of the data.

Examples:

  • Age
  • Income
  • Credit history
Dependent Variable

The dependent variable is the value we want to predict.

It is also called the target variable.

In logistic regression, this variable is categorical.

Example:

1 = High Risk, 0 = Low Risk

Training Data

Training data is the dataset used to train the Machine Learning model.

It contains examples where the correct output is already known.

During training, the model learns:

  • Patterns in the data
  • Relationships between variables
  • How to make predictions

Good training data improves accuracy and model performance.

Sigmoid Function

The sigmoid function is the core of logistic regression.

It transforms any input value into a probability between 0 and 1.

Its behavior is:

  • Large values → close to 1
  • Small values → close to 0

This allows the model to interpret outputs as probabilities.

Classification Decision

After computing the probability, the model makes a classification decision.

  • If probability ≥ 0.5 → Class 1
  • If probability < 0.5 → Class 0

This threshold can be adjusted depending on the problem.
